## Description

> A Vue.js project showcasing the capabilities of Jest testing.

## Installation

Clone this repository and run the commands:

### npm

```bash
npm install
```

### yarn

```bash
yarn install
```

## How To Use

Since this is a testing example use those commands:

### npm

```bash
$ npm test
# or in watch mode
$ npm test -- --watch
```

### yarn

```bash
$ yarn test
# or in watch mode
$ yarn test -- --watch
```
